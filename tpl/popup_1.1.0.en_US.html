WELCOME TO WP-CPROTEXT

Thanks for installing the CPROTEXT.COM WordPress Plugin.

It will allow you to use this online service for copyright protection of online texts from your WordPress admin panel.

This message will only appear after the first activation of this plugin or when an upgrade requires an important notification.

IMPORTANT NOTICE

For security concerns, the e-mail address and the password associated with your CPROTEXT.COM account will never be stored in the WordPress database. This plugin will ask for your credentials each time it will require an access to your CPROTEXT.COM account.

FIRST STEPS

  1. Check that you have a registered account on CPROTEXT.COM.
  2. Jump to the WordPress Settings menu where you will find a CPROTEXT panel.
  3. Import your CPROTEXT.COM account settings.
  4. If not available, upload the font that must be used to display your texts on this WordPress blog.
  5. Select this font as the default font.
  6. Save these modifications.

You are now ready to protect your first text by activating CPROTEXT in the editor panel before hitting the WordPress Publish button.

If you have any questions related to CPROTEXT.COM or this plugin, please first check the F.A.Q. on CPROTEXT.COM web site.
