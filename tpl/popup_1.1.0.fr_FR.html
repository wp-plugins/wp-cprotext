BIENVENUE DANS WP-CPROTEXT

Merci d'avoir installé l'extension WordPress de CPROTEXT.COM.

Cela vous permettra d'utiliser ce service en ligne pour la protection des droits d'auteurs de textes en ligne depuis votre page d'administration WordPress.

Ce message n'apparaitra qu'à la suite de la première activation de cette extension ou lorsqu'une mise à jour nécessitera une notification particulière.

REMARQUE IMPORTANTE

Pour des questions de sécurité, l'adresse e-mail et le mot de passe associés à votre compte CPROTEXT.COM ne seront jamais enregistrés dans la base de données WordPress. Cette extension vous demandera vos identifiants à chaque fois qu'elle devra accéder à votre compte CPROTEXT.COM.

PREMIERS PAS

  1. Assurez-vous d'avoir ouvert un compte sur CPROTEXT.COM.
  2. Rejoignez le menu des Réglages WordPress où vous trouverez un onglet CPROTEXT.
  3. Importez les préférences de votre compte CPROTEXT.COM.
  4. Si elle n'est pas listée, ajoutez la police de caractères qui doit être utilisée pour l'affichage de vos textes dans ce blog WordPress.
  5. Selectionnez cette police de caractères comme la police par défaut.
  6. Enregistrez ces modifications.

Vous êtes maintenant prêt à protéger votre premier texte en activant CPROTEXT dans la page d'édition avant de cliquer sur le bouton Publier de WordPress.

Pour toutes questions relatives à CPROTEXT.COM ou cette extension, consultez la F.A.Q. sur le site de CPROTEXT.COM
