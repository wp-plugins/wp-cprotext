BIENVENUE DANS WP-CPROTEXT

Merci d'avoir installé l'extension WordPress du service en ligne CPROTEXT.COM.

Le service en ligne CPROTEXT.COM permet de protéger les textes publiés sur le web contre les tentatives de copies non autorisées. Cette extension intègre ce service dans le processus de publication de votre blog WordPress de manière quasi transparente. Elle nécessite d'avoir préablablement créé gratuitement un compte sur le site CPROTEXT.COM.

REMARQUE IMPORTANTE

Pour des questions de sécurité, l'adresse e-mail et le mot de passe associés à votre compte CPROTEXT.COM ne seront jamais enregistrés dans la base de données WordPress. Cette extension vous demandera vos identifiants quand ceux-ci seront nécessaires.

PREMIERS PAS

  1. Assurez-vous d'avoir ouvert un compte gratuit sur CPROTEXT.COM.
  2. Cliquez sur l'onglet « CPROTEXT » du menu « Réglagesé » de WordPress.
  3. Importez les préférences de votre compte CPROTEXT.COM.
  4. Si elle n'est pas déjà présente dans la liste des polices disponibles, ajoutez la police de caractères qui doit être utilisée pour l'affichage de vos textes dans ce blog WordPress. (cf. F.A.Q.)
  5. Selectionnez cette police de caractères comme la police par défaut.
  6. Enregistrez ces modifications.

Après avoir rédigé votre texte dans la page d'édition de WordPress, vous pouvez demander à le protéger en activant CPROTEXT avant de cliquer sur le bouton « Publier ».

Pour toutes questions relatives au service en ligne CPROTEXT.COM ou à cette extension, consultez la F.A.Q. sur le site de CPROTEXT.COM
